# Catalog Mirror

This folder mirrors the current `ISM_WAVER` hardware-catalog package from the main EMWaver repository.

Contents:

- `device.json` - catalog metadata used by EMWaver web and app surfaces.
- `images/` - the current photo and render set referenced by that metadata.

The `images/` folder is intended to act as the canonical repo-backed image source for EMWaver catalog surfaces as the web catalog migrates away from duplicated copies under `web/public/`.
