# Videos

This folder stores video planning metadata.

## Plan and Principle

EMWaver video content here is promotional and lightweight.

- We are not optimizing for YouTuber-style content production.
- Videos should be no-audio by default: short clips, on-screen text, and background music.
- The goal is to quickly show EMWaver hardware/software flows in a clear visual way.
- Long-form tutorials and deep usage walkthroughs are intentionally out of scope here.
- Detailed tutorial-style content is better handled by creators/educators with that focus.

- Use one markdown file per produced video (for example, `promotional_video.md`).
- Each markdown file should include:
  - Drive folder name and link.
  - Source clip file names with `done`/`pending` status.
  - Date recorded for each source clip.
  - Final output file name (for example, `final.mp4`) with status.
