# Hardware

This folder holds the hardware package for EMWaver Link.

Current layout:

- `revisions/v1/fabrication/` - committed fabrication outputs for the current known revision

Current files in `revisions/v1/fabrication/`:

- `BOM_EMWaver_2026-01-21.csv`
- `CPL_EMWaver_2026-01-21.csv`
- `Gerber_EMWaver_2026-01-21.zip`
- `PCB_emwaver_2025-12-09.pdf`
- `emwaver.stl`

Intended expansion:

- `revisions/<revision>/source/` - editable EDA source files
- `revisions/<revision>/fabrication/` - fabrication exports, BOM, and pick-and-place files
- `revisions/<revision>/docs/` - revision-specific notes, pinout docs, and assembly caveats
