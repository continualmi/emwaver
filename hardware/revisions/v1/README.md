# Revision v1

Use this folder for the first committed local hardware package for EMWaver Shield.

Recommended subfolders:

- `source/` - editable EDA source
- `fabrication/` - Gerbers, BOM, pick-and-place, drill files
- `docs/` - pinouts, assembly notes, and revision-specific caveats

At the moment, the design references still live externally and are documented in [../../../docs/reference/design-sources.md](../../../docs/reference/design-sources.md).
