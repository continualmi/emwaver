## EMWaver Android App
This is the EMWaver Android App project, developed using android studio. 
EMWaver, short for Electomagnetic Waver, is a company that develops USB type C devices that connect to the Android Phone. 
Using USB 2.0, communication is established, and the user can, through the app, control various kinds of hardware. 
### ISM Waver
ISM Waver, short for Industrial, Scientific and Medical waver, is one of the devices that EMWaver offers. It is powered by a CC1101 radio chip, which allow for communications in the ISM frequency band. 
### IR Waver
IR Waver, short for Infrared Waver, is another of the devices that EMWaver offers. It posesses a infrared LED, as well as a infrared receiver, allowing to transceive any infrared signal. 
### GPIO Waver
GPIO Waver, short for General Purpose Input/Output Waver, is another device that EMWaver offers, and perehaps the most important. It posesses female headers, 18 of them, that allow the user to attach Waver Modules. 
Waver Modules are pieces of hardware with 18 male headers, that connect to the GPIO Waver, and can have various hardware capabilities. 
GPIO Waver's purpose is to enable the user to develop and interact with almost any electronics low voltage hardware, by communicating to the microcontroller thought the Android app, to various interfaces such as SPI, I2C, UART, and more.

The EMWaver Android App will be able to control all of these three devices fully.
