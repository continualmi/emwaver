# EMWaver Core

![EMWaver Core](catalog/images/EMWAVER.png)

EMWaver Core is the STM32-based EMWaver board without a built-in CC1101 radio. It keeps the compact hands-on board format with two GPIO blocks and infrared support.

This private repository starts as the device home for EMWaver Core hardware material. The current thumbnail mirrors the image used by the EMWaver web build catalog.

Current hardware files live at the repo root next to this README.
The mirrored web catalog package lives under `catalog/`.
