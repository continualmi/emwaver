# Apps Overview

EMWaver ships companion apps for Android, iOS, and desktop.

## Android

- Location: `android/`
- Responsibilities: device connection, capture workflows, wavelets UI + Git sync, and tooling parity with iOS/desktop.

## iOS

- Location: `ios/`
- Responsibilities: SwiftUI feature parity with Android (wavelets, capture tooling, Git sync).

## Desktop

- Location: `app/`
- Responsibilities: cross-platform authoring environment (editor + preview), Git repo workflows, and full hardware interaction.

## Important Notes

- iOS builds should be done in Xcode.
- Android builds should be done manually by the developer unless explicitly troubleshooting build errors.
